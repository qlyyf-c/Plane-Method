# 平法助手 - 手机端测试指南

## 方法一：浏览器开发者工具模拟

1. 打开浏览器开发者工具 (F12)
2. 点击设备模拟图标 (或按 Ctrl+Shift+M)
3. 选择移动设备 (如 iPhone 12, Pixel 5 等)
4. 测试页面响应式效果

## 方法二：手机本地测试

### 确保网络连通性

1. **确认你的 PC 和手机在同一网络**
2. **获取 PC 的局域网 IP 地址**：
   ```bash
   ip addr show | grep inet
   # 或者在 Windows 中使用
   ipconfig
   ```

3. **修改前端配置**：
   - 修改 `/frontend/vite.config.ts` 中的 `server.host` 设置为 PC 的局域网 IP
   - 或者使用 `--host` 参数启动

4. **启动服务**（在 PC 上）：
   ```bash
   # 后端（假设 IP 是 192.168.1.100）
   python run_dev.py
   
   # 前端
   cd frontend && npm run dev -- --host 0.0.0.0 --port 5173
   ```

5. **手机访问**：
   - 在手机浏览器中访问：`http://你的PC局域网IP:5173`

## 方法三：查看构建结果

由于我们已经成功构建了生产版本，您可以：

1. 检查构建结果：
   ```bash
   ls -la frontend/dist/
   ```

2. 通过本地服务器测试（需要 Python）：
   ```bash
   cd frontend/dist
   python -m http.server 8000
   ```

## 已完成的手机端优化

### 导航优化
- 底部导航栏最小高度 56px
- 图标 + 文字显示，字体 11px
- 触摸区域扩大

### 页面布局
- 所有页面使用响应式网格布局
- 手机端垂直排列，平板端水平排列
- 适当间距和边距

### 表单优化
- 输入框防自动缩放 (16px 字体)
- Parser 页面使用 textarea 支持多行输入
- 错误提示统一格式

### 性能优化
- 使用 Pinia Store 缓存数据
- 减少重复 API 请求
- 构建优化，包含 PWA 预缓存
