# 用户测试计划

## 测试目标

验证平法助手应用在实际使用场景中的可用性和功能性。

## 测试对象

3 名土木工程专业学生

## 测试内容

### 测试任务 1: 锚固计算
1. 打开应用
2. 进入锚固计算器页面
3. 输入参数：C30/HRB400/25mm/二级
4. 点击计算按钮
5. 验证结果准确性

### 测试任务 2: 标注解析
1. 进入标注解析器页面
2. 输入标注：KL7(3) 300×650
3. 点击解析按钮
4. 验证解析结果和释义

### 测试任务 3: 图集速查
1. 进入图集速查页面
2. 搜索关键词："锚固"
3. 查看搜索结果
4. 点击查看详情

### 测试任务 4: 三模块闭环
1. 从锚固计算页面点击"查看相关条文"
2. 从标注解析器页面点击"计算锚固"
3. 从图集速查页面点击"解析相关标注"

## 预期结果

- 所有功能页面可正常访问
- 计算结果准确（与22G101图集对照）
- 标注解析准确率高
- 模块间跳转顺畅
- 用户界面友好，易于理解
