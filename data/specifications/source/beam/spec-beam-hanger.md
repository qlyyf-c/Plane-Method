# 4.2.6 梁吊筋与附加箍筋

## 1. 设置条件

**应设置吊筋或附加箍筋的情况：**
- 主次梁交接处，次梁上部有集中荷载
- 框架梁承受墙体重量等较大集中荷载
- 吊车梁承受吊车轮压产生的集中荷载
- 其他需要传递集中荷载的部位

**可不设置的特殊情况：**
- 当主梁配置足够的抗剪箍筋时
- 当集中荷载较小且经计算不需要时
- 当采用其他方式（如加腋）有效传递荷载时

## 2. 吊筋构造

**吊筋形式：**
- 宜采用 HRB400 级钢筋
- 可采用单肢吊筋或双肢吊筋
- 吊筋应弯折成 45°或 60°角

**锚固长度：**
- 吊筋上端水平段锚固长度不应小于 20d
- 吊筋下端应伸至主梁底部并弯钩
- 弯钩角度不应小于 45°，平直段长度不小于 10d

**截面尺寸：**
- 吊筋直径不宜小于 12mm
- 当集中荷载较大时，直径不宜小于 16mm
- 当荷载特别大时，可采用多根吊筋组合

**布置位置：**
- 吊筋应布置在集中荷载作用位置两侧
- 第一道吊筋距离集中荷载作用点不应大于 50mm
- 吊筋间距不应大于 300mm

## 3. 附加箍筋构造

**附加箍筋直径：**
- 应与主梁箍筋直径相同
- 当主梁箍筋直径小于 8mm 时，附加箍筋直径不应小于 8mm

**附加箍筋间距：**
- 附加箍筋应在集中荷载作用范围两侧各 3s 范围内设置
- s 为加密区间距
- 间距不应大于 100mm

**附加箍筋数量：**
- 每侧不应少于 3 个
- 总数量不应少于 6 个
- 当集中荷载特别大时，应通过计算确定

**附加箍筋形式：**
- 应采用封闭式箍筋
- 箍筋应包围主梁下部纵筋
- 箍筋末端应做成 135°弯钩

## 4. 设计计算公式

**吊筋承载力：**
```
Fd ≤ 2 × fy × As × sinα
```
其中：
- Fd — 吊筋承担的集中荷载设计值
- fy — 吊筋抗拉强度设计值
- As — 吊筋截面面积
- α — 吊筋弯折角度

**附加箍筋承载力：**
```
Fv ≤ nv × fyv × Av
```
其中：
- Fv — 附加箍筋承担的集中荷载设计值
- nv — 附加箍筋肢数
- fyv — 箍筋抗拉强度设计值
- Av — 单肢箍筋截面面积

## 5. 施工注意事项

- 吊筋和附加箍筋应在绑扎主梁钢筋时同时安装
- 应先固定吊筋位置，再绑扎附加箍筋
- 吊筋下端的弯钩应紧贴主梁底筋
- 附加箍筋应与主梁箍筋绑扎牢固
- 隐蔽工程验收时应重点检查吊筋和附加箍筋的安装质量
