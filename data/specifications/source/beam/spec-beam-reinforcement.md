# 4.2.3 梁纵向钢筋配置

## 1. 上部纵筋

**通长筋：**
- 框架梁应沿梁全长设置通长纵向钢筋
- 当梁跨度大于 4m 时，直径不宜小于 10mm
- 当梁跨度大于 6m 时，直径不宜小于 12mm
- 当梁跨度大于 9m 时，直径不宜小于 14mm

**支座负筋：**
- 第一排支座负筋延伸至净跨的 1/3 处
- 第二排支座负筋延伸至净跨的 1/4 处
- 支座负筋应从柱边或梁边开始延伸

## 2. 下部纵筋

**底部纵筋：**
- 底部纵筋应在支座内锚固
- 当支座宽度满足直锚要求时，可采用直锚
- 当支座宽度不满足直锚要求时，应采用弯锚

**锚固长度：**
- 底筋在端支座的锚固长度不应小于 laE（抗震）或 la（非抗震）
- 底筋在中支座的锚固长度不应小于 laE 或 la

## 3. 侧面纵筋

**构造钢筋（G 开头）：**
- 当梁腹板高度 hw ≥ 450mm 时，应配置侧面构造钢筋
- 侧面构造钢筋间距不宜大于 200mm
- 侧面构造钢筋的直径不宜小于 12mm

**受扭钢筋（N 开头）：**
- 当梁承受扭矩时，应配置抗扭纵筋
- 抗扭纵筋应沿截面周边均匀布置
- 抗扭纵筋的间距不应大于 200mm 及梁截面短边长度
