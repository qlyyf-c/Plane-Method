# 2.3 柱节点构造

## 1. 框架梁柱节点

**节点核心区：**
- 框架梁柱节点核心区的箍筋配置应满足抗震要求
- 当梁宽大于柱宽时，应设置加腋或偏心连接
- 节点核心区高度不应小于框架梁高度

**纵向钢筋锚固：**
- 框架柱纵向钢筋在顶层边节点和角节点应弯折锚固
- 弯折段水平投影长度不应小于 0.4labE
- 弯折段竖直投影长度不应小于 15d（d 为纵筋直径）

## 2. 柱纵向钢筋连接

**连接位置：**
- 柱纵向钢筋的连接位置宜避开柱端箍筋加密区
- 当无法避开时，应采用机械连接或焊接
- 同一截面内钢筋接头面积百分率不宜超过 50%

**搭接长度：**
- 受拉钢筋搭接长度不应小于 1.2laE（抗震）或 1.2la（非抗震）
- 受压钢筋搭接长度不应小于 0.85laE 或 0.85la

## 3. 柱变截面构造

**向下变截面：**
- 当上层柱截面尺寸小于下层柱时，上层柱纵筋可向下弯折插入下层
- 弯折角度不应大于 1:6
- 插入深度应满足锚固要求

**向上变截面：**
- 当上层柱截面尺寸大于下层柱时，下层柱纵筋可按一定角度弯折向上延伸
- 弯折部分应伸入上层柱并满足锚固要求

## 4. 柱与墙体连接

- 框架柱与填充墙连接处应设置拉结筋
- 拉结筋沿墙高间距不宜超过 500mm
- 拉结筋伸入墙内的长度不应小于 700mm，且不小于墙长的 1/5
