# 2.5 柱与基础连接构造

## 1. 柱纵筋在基础中的锚固

**直锚：**
- 当基础高度满足直锚要求时，可采用直锚
- 直锚长度不应小于 laE（抗震）或 la（非抗震）
- 柱纵筋应伸至基础底部并弯折

**弯锚：**
- 当基础高度不满足直锚要求时，应采用弯锚
- 水平段长度不应小于 0.6labE
- 竖直段长度应为 15d（d 为纵筋直径）

**锚固起点：**
- 从基础顶面开始计算锚固长度
- 基础内应有混凝土保护层
- 保护层厚度不应小于 40mm

## 2. 基础顶面构造

**插筋固定：**
- 柱纵筋在基础中应采用定位箍筋固定
- 定位箍筋间距不宜大于 500mm
- 至少设置两道定位箍筋

**基础顶面处理：**
- 基础顶面应平整，不得有松散混凝土
- 浇筑上层柱混凝土前应清理基础顶面
- 必要时应进行界面处理

## 3. 变截面柱锚固

**向下变截面：**
- 上层柱纵筋可弯折插入下层柱
- 弯折角度不应大于 1:6
- 插入深度应满足锚固要求

**向上变截面：**
- 下层柱纵筋可按一定角度弯折向上延伸
- 弯折部分应伸入上层柱并满足锚固要求
- 当无法弯折时应采用机械连接

## 4. 多层基础连接

**第一层基础：**
- 柱纵筋应锚固在第一层基础中
- 锚固长度应按最不利情况考虑

**第二层及以上基础：**
- 柱纵筋可从上一层基础顶面延伸
- 延伸长度应满足搭接或连接要求
- 接头位置应相互错开

## 5. 特殊部位构造

**地下室柱脚：**
- 地下室底层柱纵筋应锚固在基础底板中
- 锚固长度应从地下室地面算起
- 应考虑地下水浮力的影响

**设备基础柱：**
- 设备基础柱应根据设备荷载确定锚固方式
- 大型设备基础柱宜采用预埋件连接
- 应考虑设备振动对锚固的影响

## 6. 施工质量控制

**钢筋绑扎：**
- 柱纵筋应垂直于基础底面
- 偏差不应超过 10mm
- 顶部标高偏差应在允许范围内

**混凝土浇筑：**
- 应分层浇筑，避免冲击钢筋
- 振捣棒不得直接接触钢筋
- 应确保钢筋周围混凝土密实

**成品保护：**
- 浇筑后应及时检查钢筋位置
- 发现位移应及时校正
- 做好钢筋防锈措施
