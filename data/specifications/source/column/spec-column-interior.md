# 2.6 柱内部构造要求

## 1. 纵筋间距与排列

**最小净距：**
- 柱纵向钢筋的净距不应小于 50mm
- 当柱截面较大时，可适当增加净距
- 最大净距不宜超过 300mm

**均匀布置：**
- 纵向钢筋应沿柱截面四角和周边均匀布置
- 角部钢筋应有箍筋或拉筋固定
- 中间钢筋应有箍筋或拉筋兼作横向支承

**多排布置：**
- 当纵筋数量较多时，可采用多排布置
- 各排钢筋应均匀分布
- 竖向间距不宜小于 25mm 且不小于 1.5d

## 2. 箍筋配置细节

**复合箍筋形式：**
- 矩形箍筋 + 拉筋组合
- 矩形箍筋 + 螺旋箍组合
- 多个矩形箍筋嵌套

**箍筋肢数确定：**
| 柱截面边长 | 建议肢数 |
|------------|----------|
| ≤400mm     | 四肢     |
| 400~600mm  | 六肢     |
| >600mm     | 八肢及以上 |

**拉筋设置：**
- 拉筋应同时钩住纵向钢筋和箍筋
- 拉筋直径不应小于 6mm
- 拉筋间距不应大于箍筋间距的 2 倍

## 3. 保护层厚度控制

**保护层垫块：**
- 应采用高强度混凝土垫块或塑料垫块
- 垫块强度不应低于构件混凝土强度
- 每平方米不少于 4 个垫块

**保护层厚度允许偏差：**
- 柱保护层厚度：±5mm
- 垫块位置应错开布置
- 垫块应固定在纵筋上，不得脱落

## 4. 钢筋连接质量

**机械连接：**
- 接头处应密贴，无间隙
- 外露丝扣不得超过 2 扣
- 应进行扭矩检查

**焊接连接：**
- 焊缝应饱满，无夹渣、气孔
- 焊缝长度应符合规范要求
- 焊后应清除焊渣

**绑扎搭接：**
- 搭接长度应满足计算要求
- 绑扎点不应少于 3 个
- 铁丝头应向内弯曲，不得突出保护层

## 5. 特殊节点处理

**柱变截面节点：**
- 变截面处应加密箍筋
- 箍筋间距不应大于 100mm
- 纵筋弯折角度应符合要求

**柱梁节点：**
- 节点核心区应全高加密箍筋
- 箍筋应穿过梁纵筋
- 必要时采用专用节点箍筋

**柱墙连接：**
- 柱纵筋应锚入墙内
- 锚固长度应满足要求
- 连接处应设置附加钢筋

## 6. 施工检验

**外观检查：**
- 钢筋规格、数量、间距应符合设计
- 保护层厚度应符合要求
- 连接接头质量应合格

**实测数据：**
- 纵筋间距偏差≤10mm
- 保护层厚度偏差≤5mm
- 柱垂直度偏差≤H/1000 且≤30mm

**隐蔽验收：**
- 钢筋工程验收合格后才能浇筑混凝土
- 验收记录应完整真实
- 影像资料应妥善保存
