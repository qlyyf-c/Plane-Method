# 2.1 柱平法施工图的表示方法

柱平法施工图系在柱平面布置图上采用列表注写方式或截面注写方式表达。

**列表注写方式：**在柱平面布置图上，分别在同一编号的柱中选择一个或几个截面标注几何参数代号；在柱表中注写柱编号、柱段起止标高、几何尺寸（含截面对轴线的偏心情况）与配筋的具体数值。

**截面注写方式：**在柱平面布置图的柱截面上，分别在同一编号的柱中选择一个或几个截面直接注写截面尺寸和配筋具体数值。

**柱编号规则：**

| 代号 | 名称 |
|------|------|
| KZ   | 框架柱 |
| Z    | 非框架柱 |
| QZ   | 剪力墙上柱 |
| LZ   | 梁上柱 |
| ZH   | 转换柱 |
| XZ   | 芯柱 |

**柱截面尺寸标注：**矩形柱：b×h（宽×高）；圆柱：d（直径）
