# 11.2 梁柱节点构造

## 1. 节点核心区尺寸

**矩形节点：**
- 节点的宽度不应小于柱宽
- 节点的有效高度不应小于框架梁的高度
- 当梁宽与柱宽之差小于柱宽的 1/4 时，可采取加腋措施

**偏心节点：**
- 当梁中心线与柱中心线不重合时，偏心距不宜大于柱截面该方向尺寸的 1/4
- 当偏心距超过限值时，应采取加强措施或进行专门设计

## 2. 节点箍筋配置

**加密要求：**
- 框架梁柱节点核心区的箍筋应全高加密
- 箍筋间距不应大于 100mm
- 一级抗震时，箍筋直径不应小于 10mm；二~四级不应小于 8mm

**体积配箍率：**
| 抗震等级 | 角节点   | 边节点   | 中节点   |
|----------|----------|----------|----------|
| 一级     | 1.2%     | 1.0%     | 0.8%     |
| 二级     | 1.0%     | 0.8%     | 0.6%     |
| 三级     | 0.8%     | 0.6%     | 0.5%     |
| 四级     | 0.6%     | 0.5%     | 0.4%     |

## 3. 纵向钢筋锚固

**框架梁上部纵筋：**
- 应贯穿节点或按锚固要求弯折
- 直线锚固长度不应小于 labE（抗震）或 lab（非抗震）
- 弯折锚固时，水平段长度不应小于 0.4labE，竖直段长度不应小于 15d

**框架梁下部纵筋：**
- 端支座处应满足锚固要求
- 可采用直锚或弯锚形式
- 中支座处应按受拉钢筋锚固

**框架柱纵筋：**
- 应贯通节点核心区
- 顶层边柱和角柱纵筋应弯折锚固
- 弯折段水平投影≥0.4labE，竖直投影≥15d

## 4. 节点核心区受剪承载力

节点核心区的受剪承载力应符合下列公式要求：

```
Vj ≤ (0.1ηfcbcjhj + 0.9fyvAvsj(hj0-as')/s) / γRE
```

其中：
- Vj — 节点核心区剪力设计值
- η — 约束修正系数
- fc — 混凝土轴心抗压强度设计值
- bcb、hj — 节点核心区的计算宽度和有效高度
- fyv、Avsj、s — 箍筋抗拉强度、总截面积和间距

## 5. 特殊节点处理

**转换节点：**
- 转换梁上柱的节点应按特殊节点设计
- 应进行详细的受力分析和构造处理
- 必要时进行试验验证

**复杂节点：**
- 多向梁交汇的节点应进行专门分析
- 应考虑各方向梁对节点的共同作用
- 必要时采用有限元方法进行验算
